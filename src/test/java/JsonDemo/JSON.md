**org.json:JSON官方推荐的解析类**
    --简单易用，通用性强
    --复杂功能欠缺
**GSON：Google的**
    --基于反射，可以实现JSON对象、json字符串和java对象的转换
**Jackson:号称最快的JSON处理器**
简单易用、社区更新发布速度快
 
 
 网上的信息
 1.传统Json方式：
    复杂的Json数据转换成实体类存在缺陷，性能和功能不够完善
 2.Jackson方式：
    复杂的Json数据转换成实体类存在缺陷，性能和功能优于传统方式
 3.Gson方式： 
    功能方面在几种方式中最优，性能方面不如Jackson方式
 4.FastJson方式：
    复杂的实体类转换成Json数据存在缺陷，解析json的速度优于其他方式
